**Crée un site web pour une agence digitale (nom: "Digital Nexus") en français/arabe avec ces exigences :**  

1. **Objectif Principal**  
   - Convertir les visiteurs en leads via des formulaires de contact optimisés.  
   - Renforcer la crédibilité avec des preuves sociales (témoignages, études de cas).  

2. **Design & UX**  
   - Palette : Bleu royal (#2A5EE8) et dégradés de gris foncé (#1E1E2C) + touches de vert (#00C897) pour les CTA.  
   - Typographie : Inter pour le corps, Montserrat pour les titres (Google Fonts).  
   - Animations : Micro-interactions au survol (boutons, cartes), défilement parallaxe.  

3. **Sections Obligatoires**  
   - Hero Section : Vidéo d'arrière-plan + slogan percutant ("Transformez Votre Vision en Réalité Digitale").  
   - Services : Grille interactive (3 colonnes) avec icônes SVG personnalisées.  
   - Portfolio : Filtres par catégorie (création site web, Formation whatsapp Business, brandingL ) + modaux détaillés.  
   - Témoignages : Carrousel auto-défilant avec photos clients et logos d'entreprises.  
   - Contact : Formulaire intelligent en 3 étapes (coordonnées+ besoins + moyens de contact) avec intégration HubSpot/Netlify.  

4. **Optimisations Techniques**  
   - SEO : Balises méta dynamiques (via Next.js), schema.org pour les services, blog intégré (Headless CMS).  
   - Performance : Lazy loading des images, compression WebP, préchargement des polices.  
   - Accessibilité : Niveau AA (contraste vérifié via Lighthouse), navigation au clavier.  

5. **Fonctionnalités Avancées**  
   - Chat en direct (Tawk.to ou Crisp) pour le support instantané.  
   - Calculateur de budget interactif (sliders pour sélectionner les services).  
   - Intégration LinkedIn/Instagram pour un flux "actualités" en temps réel.  

6. **Stack Technique**  
   - Framework : Next.js (SSR pour le SEO).  
   - Style : Tailwind CSS + Framer Motion pour les animations.  
   - Déploiement : Vercel (optimisations intégrées).  

7. **Livrables Attendus**  
   - Maquettes Figma (version desktop/mobile) incluant les états interactifs (hover, focus).  
   - Code source commenté + documentation d'installation.  
   - Audit SEO initial via Screaming Frog.
